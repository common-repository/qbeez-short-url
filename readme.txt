== Description ==

When you create/update a page or post the plugin will check if a QBeez has been created or not. If not the plugin will create a new one. Use your own QBeez API key to load the newly created QBeez into your private account.

== Installation ==

1. Upload the complete folder `qbeez-short-url` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to 'settings' -> 'QBeez Settings' and choose which domain you want to use for shorting your page/post url.

4. Optional: Change API key with your own private API key so newly create QBeez will be loaded into your qbeez.nl account.
